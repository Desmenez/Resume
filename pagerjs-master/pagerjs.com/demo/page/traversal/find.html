Find

Finding pages by name can be done using pager.Page.prototype.find or pager.Page.prototype.find$. The string can be an absolute or relative path. 

// adminPage is a pager.Page
var adminPage = pager.page.find('admin');
// loginPage is an Observable(pager.Page). Useful in bindings.
var loginPage = pager.page.find$('user/login');
